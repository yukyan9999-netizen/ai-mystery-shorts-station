# SFX (Sound Effects) Library

Place `.mp3` or `.wav` sound effect files in this folder.

They will appear in the dashboard scene editor as selectable SFX options.
Each SFX file can be assigned to individual scenes to play alongside the narration audio.

## Volume

The default SFX volume is controlled by `sfx_library.default_volume` in `config.yaml` (default: 0.3).

## Naming Convention

Use descriptive filenames, e.g.:
- `thunder.mp3`
- `whoosh.mp3`
- `suspense_hit.mp3`
- `clock_ticking.wav`

The filename (without extension) is displayed as the SFX name in the dashboard.
